# OverlayD3D9
Simple overlay using `D3D9`

# Requirements
Only supports win10 x64

# Features
- Draw Strings  
L Custom Colors  
L Auto Centering  
L Outline For Visibility  
- Draw Lines  
L Custom Colors  
- Draw Rectangles  
L Custom Colors  
L Outline For Visibility  
- Draw Circles  
L Custom Colors  
- String Xoring
